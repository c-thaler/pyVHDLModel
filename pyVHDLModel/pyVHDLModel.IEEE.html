pyVHDLModel.IEEE¶

This module contains library and package declarations for VHDL library IEEE.

Classes

  • Ieee: A Library represents a VHDL library. It contains all primary and secondary design units.

  • Math_Real: A PrimaryUnit is a base-class for all primary units.
