pyVHDLModel.Object¶

This module contains parts of an abstract document language model for VHDL.

Objects are constants, variables, signals and files.

Classes

  • Obj: ModelEntity is the base-class for all classes in the VHDL language model, except for mixin classes (see multiple

  • BaseConstant: ModelEntity is the base-class for all classes in the VHDL language model, except for mixin classes (see multiple

  • WithDefaultExpressionMixin: A WithDefaultExpression is a mixin class for all objects declarations accepting default expressions.
