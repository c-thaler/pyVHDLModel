pyVHDLModel.Instantiation¶

This module contains parts of an abstract document language model for VHDL.

Instantiations of packages, procedures, functions and protected types.

Classes

  • GenericEntityInstantiation: Undocumented.

  • SubprogramInstantiation: ModelEntity is the base-class for all classes in the VHDL language model, except for mixin classes (see multiple

  • ProcedureInstantiation: ModelEntity is the base-class for all classes in the VHDL language model, except for mixin classes (see multiple

  • FunctionInstantiation: ModelEntity is the base-class for all classes in the VHDL language model, except for mixin classes (see multiple

  • PackageInstantiation: A PrimaryUnit is a base-class for all primary units.
