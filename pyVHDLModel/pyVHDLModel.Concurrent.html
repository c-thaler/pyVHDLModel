pyVHDLModel.Concurrent¶

This module contains parts of an abstract document language model for VHDL.

Concurrent defines all concurrent statements used in entities, architectures, generates and block statements.

Classes

  • ConcurrentStatement: A ConcurrentStatement is a base-class for all concurrent statements.

  • ConcurrentStatements: Undocumented.

  • Instantiation: A ConcurrentStatement is a base-class for all concurrent statements.

  • ConcurrentProcedureCall: A ConcurrentStatement is a base-class for all concurrent statements.

  • ConcurrentDeclarations: Undocumented.

  • ConcurrentBlockStatement: A ConcurrentStatement is a base-class for all concurrent statements.

  • ElsifGenerateBranch: A GenerateBranch is a base-class for all branches in a generate statements.

  • GenerateStatement: A GenerateStatement is a base-class for all generate statements.

  • IfGenerateStatement: A GenerateStatement is a base-class for all generate statements.

  • ConcurrentChoice: A ConcurrentChoice is a base-class for all concurrent choices (in case…generate statements).

  • IndexedGenerateChoice: A ConcurrentChoice is a base-class for all concurrent choices (in case…generate statements).
